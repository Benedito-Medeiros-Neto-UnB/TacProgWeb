Bem-vindo ao Portal Jornalismo de Dados!


"O Jornalismo de Dados é uma modalidade de produção digital de notícias que utiliza grandes bases de dados para elaborar peças de conteúdo focadas em correlações informacionais, além de se valer de recursos gráficos e interativos para tornar mais agradável a experiência de visualização do usuário consumidor de notícias." - Érico Mafra


  • Este portal foi desenvolvido com o objetivo de fornecer um centro de referência para alunos e demais interessados na grande e emergente área de Jornalismo de Dados!

  • Além de referências científicas e de portais na internet, este portal também contem uma ferramenta para auxilar entusiastas na coleta de dados do Twitter e uma ferramenta para gerar infográficos.
