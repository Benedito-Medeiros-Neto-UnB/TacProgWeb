Linguagem de programação R.

Para que serve o R?

R é uma linguagem de programação multi-paradigma, ou seja, podemos utilizar princípios tanto do paradigma funcional, sequencial e o orientado a objetos.
Porém o R brilha mesmo é na parte estátistica, com pacotes que facilitam muito na hora de desenvolver. Aqui nesse site vou abordar principalmente sobre o ggplot2 e sobre o dplyr.

R Studio

Para melhorar sua experiência com a linguagem R recomendamos a utilização do ambiente de desenvolvimento R Studio.
Conseguimos aproveitar melhor dos pacotes para R dentro dessa IDE, por possuir uma parte para download de packages e até visualização dos dados que você importar.

Ggplot2

Ggplot2 é um pacote para visualização de dados dentro da linguagem estátistica R.

Onde o intuito é gerar gráficos de maneira rápida e fácil. Baseado no The Grammar of Graphics, livro de estátistica na computação, o ggplot2 recebe uma entrada de dados e com essa entrada podemos manipular utilizando a biblioteca.

Instalação

Podemos installar da maneira mais fácil que é utilizando o tidyverse:

install.packages("tidyverse")

Com esse pacote teremos praticamente todas as ferramentas necessárias para manipular dados dentro do R Studio. Ou também podemos installar individualmente:

install.packages("ggplot2")

E também temos mais uma maneira que será explicado no vídeo no final dessa seção.

Utilização

Bom, o pacote ggplot2 possui muitas funcionalidades para apenas descrever aqui em uma Wiki, porém, temos comandos que quase sempre iremos utilizar, como por exemplo, o comando ggplot() que seguido pode ser combinado com o comando: aes() que serve pra declararmos a estética do nosso gráfico. Exemplo:

library(ggplot2) ggplot(mpg, aes(displ, hwy, colour = class)) + geom_point()

Esse exemplo foi retirado do site do próprio tidyverse que temos aqui: link.

Dplyr

Dplyr é uma biblioteca, ou também chamado de pacote(package), que nós da acesso a maneiras mais fáceis de manipular os dados.

  • select() pegamos uma variável de acordo com o seu nome.
  • filter() filtramos os nossos dados baseado em uma condicional.
  • summary() reduzimos os nossos dados para apenas uma variável.
  • groub_by() onde podemos realizar uma operação para determinado grupo(group).
Instalação

Para instalar o dplyr seguimos a mesma ideia do ggplot2. Assim executando os seguintes comandos no nosso RStudio:

install.packages("tidyverse") install.packages("dplyr")

Onde usando o tidyverse podemos instalar várias ferramentas de uma vez, ou instalar individualmente essa ferramenta.
